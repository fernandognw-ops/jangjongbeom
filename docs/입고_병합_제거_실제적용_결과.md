# 입고 병합 제거 - 실제 적용 결과 보고

## 실행 순서 (체크리스트)

1. ☐ Supabase SQL Editor에서 `scripts/migrate_inbound_no_merge.sql` 실행
2. ☐ Production 웹(https://jangjongbeom.vercel.app)에서 동일 Excel 파일 업로드
3. ☐ 업로드 전 검증 수치 확인
4. ☐ 수치 일치 시 DB 반영 클릭
5. ☐ `node scripts/run_outbound_migration.mjs --verify` 실행
