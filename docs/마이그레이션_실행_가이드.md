# 표준 필드 마이그레이션 실행 가이드

## 1. SQL 마이그레이션 (Supabase SQL Editor)

1. [Supabase Dashboard](https://supabase.com/dashboard) → 프로젝트 선택
2. **SQL Editor** 메뉴 클릭
3. `scripts/migrate_standard_fields.sql` 파일 내용 전체 복사 후 붙여넣기
4. **Run** 실행

```sql
-- scripts/migrate_standard_fields.sql 내용 실행
```

## 2. 데이터 덮어쓰기 (Full Reset)

```bash
python scripts/upload_excel_full_reset.py
```

## 3. 대시보드 확인

```bash
npm run dev
```

브라우저에서 http://localhost:3007 접속 후 **재고 금액 871,241,076원** 표시 확인.
