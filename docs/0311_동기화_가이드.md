# 0311 수불 기준 동기화 가이드

## 개요
- **품목 수**: 478건 (0311 Rawdata 기준 현재 운영)
- **총 재고 자산**: 775,564,288원 (0311 재고 시트 기준)
- **단종 품목**: DB에 있으나 0311 Rawdata에 없는 품목

## 1. DB 스키마 적용 (필수)

**Supabase 대시보드 → SQL Editor → New query** 에서 아래 파일 전체 내용을 붙여넣고 **Run** 실행:

```
supabase-schema-stock-snapshot.sql
```

생성되는 항목:
- `inventory_products.is_active` 컬럼
- `inventory_current_products` 테이블 (현재 운영 478건)
- `inventory_stock_snapshot` 테이블 (재고 스냅샷)

## 2. 0311 동기화 스크립트 실행

```powershell
cd C:\Users\pc\inventory-system

# 환경변수 설정 (이미 되어 있으면 생략)
$env:SUPABASE_URL="https://xxxxx.supabase.co"
$env:SUPABASE_KEY="eyJ..."

# 0311 파일 기준 동기화
python scripts/sync_0311_current.py "C:\Users\pc\Desktop\장종범\인수 인계서\물류 재고 관리 시스템 구축\수불 마감 자료\0311_생산수불현황.xlsx"
```

## 3. 대시보드 변경 사항

- **현재 운영 (478건)**: 0311 Rawdata에 있는 품목만 (기본)
- **단종 품목**: 과거 운영했으나 0311에 없는 품목
- **전체**: 모든 품목
- **총 재고 자산**: 0311 재고 스냅샷 기준 775,564,288원

## 4. 이후 월별 수불 반영

새 수불 파일이 있으면:
1. `upload_excel_to_supabase.py`로 입출고 업로드
2. `sync_0311_current.py`를 **새 파일**로 다시 실행하여 현재 품목·재고 스냅샷 갱신
