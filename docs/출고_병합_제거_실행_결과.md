## 1. 마이그레이션 실행 여부

| 항목 | 상태 |
|------|------|
| **실행 방법** | Supabase SQL Editor 수동 또는 `node scripts/run_outbound_migration.mjs --execute` (DATABASE_URL 필요) |
| **실행 여부** | ☐ 완료 (사용자 확인 필요) |

**실행 방법**:
- Supabase 대시보드 → SQL Editor → `scripts/migrate_outbound_no_merge.sql` 내용 붙여넣기 → Run
- 또는 `.env.local`에 `DATABASE_URL`(Supabase Connection string) 추가 후 `node scripts/run_outbound_migration.mjs --execute`
