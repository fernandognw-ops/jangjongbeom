# UNIQUE 제약 조건 롤백 가이드

## 1. 즉시 실행 (Supabase SQL Editor)

`scripts/rollback_unique_constraints.sql` 파일 내용을 **Supabase Dashboard → SQL Editor**에 붙여넣고 실행하세요.

- `unique_outbound_record`, `unique_inbound_record`, `unique_current_stock` 제거
- `idx_inbound_upsert`, `idx_outbound_upsert` 제거
- 채널 비어있는 과거 데이터 → `general`로 업데이트
- `sales_channel` 기본값 `general` 설정

## 2. 데이터 재적재 (선택)

제약 때문에 이전에 실패한 업로드가 있다면:

- **엑셀 업로드**: 생산수불현황.xlsx 다시 업로드
- **스크립트**: `python scripts/sync_0311_current.py [엑셀경로]` 재실행

## 3. 재고 금액 확인

- 대시보드 새로고침 후 **재고 금액**이 억 단위로 표시되는지 확인
- 단가가 0인 품목은 `inventory_products.unit_cost`로 보완됨
