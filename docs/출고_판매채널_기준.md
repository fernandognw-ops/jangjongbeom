# 출고 채널 기준 (판매 채널 통일)

## 수정 전

- 출고 시트에서 `sales_channel` 동의어 **「매출구분」「판매처」** 등으로 열을 잡거나, **판매 채널 셀이 비면 출고처·보관센터**로 채널을 추론함.

## 수정 후

- 출고 시트 **「판매 채널」** 열만 사용 (`outbound_sales_channel` 동의어: 판매 채널, 판매채널, 판매 채널명).
- 해당 열을 찾지 못하면 파싱 0건 + 검증 **400** (DB 반영 불가).
- `dest_warehouse` / `warehouse_group` / DB `sales_channel`은 모두 이 열 값을 `normalize_sales_channel_kr` / `ensureChannel`로 정규화한 결과로 통일.

## 검증 3way

| 단계 | 기준 |
|------|------|
| 엑셀 | 시트에서 **판매 채널** 열 기준 수량 합 |
| DB | `SELECT sales_channel, SUM(quantity) FROM inventory_outbound ... GROUP BY 1` |
| 대시보드/API | `inventory_outbound.sales_channel` 및 `category-trend`의 쿠팡/일반 분해 |

총 출고 수량은 동일하고, **채널별 수량**만 엑셀 「판매 채널」과 일치해야 합니다.
