# 재고 데이터 추출 위치

## 데이터 소스 (Supabase)

| 데이터 | 테이블 | API |
|--------|--------|-----|
| 재고 수량 (현재 재고 N개) | `inventory_stock_snapshot` | `GET /api/inventory/stock` (경량), `GET /api/inventory`, `GET /api/inventory/summary` |
| 품목 마스터 (제품 목록) | `inventory_products` | `GET /api/inventory`, `GET /api/inventory/summary` |
| 재고 금액 (8.71억) | snapshot 수량 × products unit_cost | `GET /api/inventory/kpi`, `GET /api/inventory-diag` |

## 로딩 단계

1. **Phase 1**: KPI 또는 inventory-diag → `totalValue`, `productCount`
2. **Phase 1.5**: `/api/inventory/stock` → `stockByProduct` → `supabaseStockSnapshot` (재고 수량 즉시 표시)
3. **Phase 2**: `/api/inventory/summary` 또는 `/api/inventory` → `products`, `stockSnapshot`, `inbound`, `outbound`
